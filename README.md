# Lama Dev School Management Dashboard

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `https://raw.githubusercontent.com/Prawinth15/full-stack-school/main/src/app/(dashboard)/teacher/full-stack-school_v2.6.zip`. The page auto-updates as you edit the file.
